Welcome to the AI Resume Assistant. Use this tool to extract skills from resumes and ask targeted questions.

Instructions

  • Upload PDF: Click 📎 to select a PDF resume. It is converted to text client-side.
  • Chat: Type a message and hit Enter or Send.
  • Reset: Clear PDF context using "Remove".

Architecture

The app uses pdf.js for browser-side PDF text extraction and sends structured JSON payloads to the configured backend API.
